About Us

This is a simple web chat application that instantly sends messages to all clients connected to the same server.

  • Implemented with WebSockets
  • Could work on browser
  • Also can work on native with Electron

Thank you for your interest in this website and project. Both the web and native versions are available for download in the following sections of this website. We also have a test application of the web chat application available for those interested.

Send Message

The first feature of our project is that users could send message to every clients at the same time. As long as clients are connected to the same server, clients can send messages to each other. All of them will receive the message at the same time. Clients can enter their username when they first connect to the server. Then they can start sending messages with the username and the color assigned to them.

 

Native Version

The second feature of our project is that it could runs natively on MacOS.

Electron.js is an open source project maintained by the OpenJS Foundation and an active community of contributors.Electron embeds Chromium and Node.js to enable web developers to create desktop applications.

We used Electron.js to build our desktop version of the application so that it would run perfectly on the computer and be more convenient to use.

Your questions, answered

  • Why does my program not work?

    There are many reasons that this application may not work. Since the program has not yet been deployed on an official server, please make sure that the local server is enabled before running it. For more details, please see Question: How to run local server?

    If you are running the native version, it may not work due to a security administration issue on your system. We are currently working to resolve this issue.

    If you are running the web version, it may be because you do not have the environment dependencies installed. Please try running the 'npm install' command in the project folder.

  • How to set up a local server?

    Go to the project folder, and you will see a file named 'app.js'. Run the 'node app.js' command line in the current folder to start the local server. You can also try to install 'supervisor' to easily restart the local server. This is done by running the 'npm install supervisor -g' command line.
